# 📚 Platform Documentation & Onboarding

Welcome to the Placement Assessment Platform. This folder contains comprehensive guides to help different users get started quickly.

### 1. [🎓 Faculty Guide](./FACULTY_GUIDE.md)

- How to onboard students in bulk (Excel/CSV).
- How to create and schedule assessments.
- How to monitor live exams and export results.

### 2. [🛡️ Admin Guide](./ADMIN_GUIDE.md)
**Best for:** System Administrators and IT Staff.
- Platform monitoring and health checks.
- Disaster recovery (Backup & Restore).
- Scaling for hundreds or thousands of students.

### 3. [💻 Student Guide](./STUDENT_GUIDE.md)
**Best for:** Candidates taking the assessment.
- Login instructions.
- Using the IDE and running/submitting code.
- Managing time and auto-submission.

Before your first real-world exam, please ensure you have performed a **Restore Test** as described in the Admin Guide to verify your data safety net.
